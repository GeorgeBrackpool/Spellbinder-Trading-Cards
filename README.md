# Spellbinder Trading Cards
Spellbinder Trading Cards is an ASP.NET Core MVC web application that simulates an online trading card store. The project was built to strengthen my full-stack development skills while demonstrating modern web development practices using C#, Entity Framework Core, SQL Server, and ASP.NET Core MVC.

## Features
Browse trading cards by game
View detailed card information
User authentication and authorization with ASP.NET Identity
Code First database design with Entity Framework Core
Database migrations and data seeding
Responsive user interface built with Bootstrap
